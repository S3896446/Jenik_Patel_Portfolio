LongD Project

Overview

As I mentioned last time, I had the opportunity to work with Quake Tech Inc as work integrated learning at RMIT. We had the opportunity to work on two projects One being Save Up / Ewyzly which was a money saving finacial app and other being LongD which was a long distance travel app similar to Uber.

The advantage we had for this project was that it was to be made using Ionic and Angular, and we had a lot of practice with Ionic and Angular last time with Save Up Project.

Problem

During this Project we didn't have many communication problems as this was the second project, however one of the problem we did encounter was that for this project we had to no only develop the app but to also use firebase to implement the database hosting and authentication for the users. As this was a first time for us we had a lot of challenges.

Solution

So to over come this issue what I did was that that responsibility on myself as it was my turn to lead the group, we had to take turn leading the group as per our assignment. However so after I took the responsibility for firebase, I began looking at tutorials and documents. To learn how to implement firebase with ionic and angular, I also got help from James as he had prior experience with implementing firebase. So throught countless nights and hard work I was able to implement firebase in out app.
